# Weiterführende Beispiele

## CSS Zen Garden

CSS Zen Garden ist ein klassisches Projekt, das die Macht von CSS demonstriert.
Die HTML-Struktur bleibt **identisch**, aber durch unterschiedliche CSS-Dateien
entstehen völlig verschiedene Designs.

## Lernziele

- Verstehen, wie CSS das Erscheinungsbild komplett verändert
- Fortgeschrittene CSS-Techniken erkunden
- Inspiration für eigene Designs sammeln

## Dateien

- `index.html` - Die HTML-Struktur (wird nicht verändert!)
- `css/style.css` - Das Stylesheet (hier passiert die Magie!)
- `js/script.js` - Optional: JavaScript für Interaktivität

## Das Konzept

Das Original CSS Zen Garden (http://www.csszengarden.com/) zeigt hunderte
verschiedene Designs - alle basierend auf demselben HTML!

**Ihre Aufgabe:**
1. Öffnen Sie `index.html` im Browser
2. Schauen Sie sich das aktuelle Design an
3. Experimentieren Sie mit `css/style.css`
4. Verändern Sie Farben, Layouts, Schriften
5. Erstellen Sie Ihr eigenes, einzigartiges Design!

## Regeln

- ❌ Ändern Sie NICHT die HTML-Struktur
- ✅ Ändern Sie NUR das CSS
- ✅ Verwenden Sie moderne CSS-Features
- ✅ Seien Sie kreativ!

## Inspiration

Besuchen Sie http://www.csszengarden.com/ für Inspiration und um zu sehen,
was alles mit CSS möglich ist.

## Fortgeschrittene CSS-Techniken

Experimentieren Sie mit:
- CSS Grid für komplexe Layouts
- CSS Animations & Transitions
- CSS Custom Properties (Variablen)
- Pseudo-Elemente (::before, ::after)
- CSS Filters & Blend Modes
- Responsive Design mit Media Queries

## Herausforderung

Erstellen Sie mindestens **drei verschiedene Designs** für dieselbe HTML-Seite:
1. Ein minimalistisches Design
2. Ein buntes, verspieltes Design
3. Ein elegantes, professionelles Design

Speichern Sie jedes Design in einer separaten CSS-Datei:
- `css/minimal.css`
- `css/playful.css`
- `css/elegant.css`
