Was ist CSS?

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die verwendet wird, um das Aussehen und Layout von HTML-Dokumenten zu beschreiben.

Wichtige Konzepte

Mit CSS können Sie Farben, Schriften, Abstände und Layouts definieren.
